# SCOUNDREL: A 52-card roguelike dungeon crawler!
A little game made with PyGame.

# Basic Rules
- Progress through an endless set of floors and encounter rooms of 4 cards at a time.
- Hearts are potions, Diamonds are weapons and Clubs & Spades are monsters.
- Defeat the entire deck of cards to move on to the next floor.
- More rules are accessible from the main menu.
